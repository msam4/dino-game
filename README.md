# Dino Game
Recreated the dino game from Google Chrome!
<br>
Press any key to start. 
<br>
Avoid the cactus!
<br>
Press space to jump!
<br>
![image](https://github.com/msam4/dino-game/assets/137851066/c0a0a0c3-9560-4b17-9527-9ac058e4c88d)

## Getting Started
### Setup

No setup is required! 

### Run a server
```
serve
```
